# 嵌入式最小系统课程设计

## 介绍

一款基于HDG12864F-1液晶显示屏的射击小游戏。<br>

本课程设计是参考于童年时期在红白机上玩过的一款射击小游戏来设计的，该射击游戏是在一定时间内统计玩家击中目标的数量来计算分数，在指定的时间内，击中目标的次数越多，则分数越高。

## 元器件

使用到的元器件如下：

- ARM : LPC2124
- LCD : HDG12864F-1

## 仿真环境

- Proteus
- Ads v1.2
